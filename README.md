# Mini E-Commerce Store

A simple full-stack e-commerce web application where users can browse products, add items to the cart, and maintain cart data using localStorage. This project helped me understand state persistence, shopping cart logic, and frontend-backend communication using Node.js and Express.js.

---

## Features

- Display product listings dynamically
- Add products to shopping cart
- Cart item count updates in real time
- Save cart data using localStorage
- Order simulation and checkout functionality
- Responsive UI using Bootstrap 5

## How to Run the Project

### 1. Clone the Repository

git clone <your-repository-link>

### 2. Open Project Folder

cd Mini-E-Commerce-Store

### 3. Install Dependencies

npm install express cors

### 4. Start Backend Server

node server.js

Server will run on:

http://localhost:8080

### 5. Run Frontend

Open `index.html` in your browser.
